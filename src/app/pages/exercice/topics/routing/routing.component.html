Créez 2 composants ainsi que les routes associées afin de les afficher dans cette page. Le premier composant peut être indépendant avec un rendu libre mais le second devra récupérer une ou plusieurs informations (par path ou data) et les utiliser dans le rendu du composant.

La zone ci-dessous recevra les liens vers les composants listés dans la liste prévue à cet effet dans le fichier app-routing.module.ts
